# Axios Sistema de Gestao

Sistema de gestao de consorcios da empresa Axios Maringa.

## Como rodar

1. Instalar dependencias:

```
npm install
```

2. Iniciar o servidor:

```
npm start
```

3. Abrir no navegador:

```
http://localhost:3000
```

## Modulos

- Estoque de Cotas
- Controle de Vendas
- Fluxo de Caixa
- Gestao de Investidores

## Tecnologias

- Node.js
- Express
- HTML e CSS

## Deploy

Hospedado no Railway. O deploy e feito automaticamente pelo GitHub.
